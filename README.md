# 💖 Maira Memon — Frontend Developer Portfolio

A modern, fully responsive personal portfolio website built with HTML, CSS, and JavaScript. Features a sleek dark/light theme, animated UI, AI chatbot assistant, and multiple interactive sections.

---

## ✨ Features

- **Animated Hero Section** — Typing effect with dynamic role & bio text
- **Dark / Light Mode Toggle** — Smooth theme switching with persistent UI
- **Project Showcase** — Filterable and searchable project grid with category tabs
- **Skills Section** — Animated progress bars for HTML, CSS, JavaScript, and React
- **Certifications** — Displayed from Meta (Coursera), freeCodeCamp, and Google
- **Testimonials Slider** — Auto-scrolling client reviews carousel
- **Tech Blog** — Articles on CSS Grid, JavaScript performance, and React basics
- **Contact Form** — Powered by [Web3Forms](https://web3forms.com/) with real-time submission feedback
- **AI Chatbot** — Built-in "Pink AI Assistant" for visitor Q&A
- **Particle Background** — Animated canvas particle effect
- **Scroll Animations** — Reveal-on-scroll effects across all sections
- **Floating Social Buttons** — Share on LinkedIn, Twitter, and WhatsApp
- **Stats Counter** — Animated counters for projects, clients, experience, and certifications
- **PDF Download** — Save portfolio / resume as PDF
- **Newsletter Signup** — Email subscription in the footer
- **Fully Responsive** — Mobile-first design with hamburger navigation
